### How to install
1. [Node.js][nodelink] 설치     
2. [Yarn][yarnlink] 설치
3. ```git clone https://github.com/seokyoung-kim/crewtocol-client```

### How to Start
1. ```cd crewtocol-client```
2. yarn install
3. yarn start
